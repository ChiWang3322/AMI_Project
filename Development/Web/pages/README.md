## Usage
In order to run this demo, you need to install `streamlit_lottie` and `streamlit` with pip command:

    1. pip install streamlit
    2. pip install streamlit_lottie
Then run these command below:

    # vanilla terminal
    streamlit run app.py

    # quit
    ctrl+c
